# 기여 가이드

## 개발 환경 설정

1. Node.js 18+ 및 pnpm 설치
2. 의존성 설치: `pnpm install`
3. 환경 변수 설정 (각 서비스의 `.env.example` 참고)

## 코드 스타일

- TypeScript strict mode 사용
- Prettier 및 ESLint 준수
- Feature-based 폴더 구조 유지

## 커밋 메시지

- `feat: 새로운 기능`
- `fix: 버그 수정`
- `refactor: 리팩토링`
- `docs: 문서 업데이트`

## Pull Request

- 기능별로 작은 단위로 PR 생성
- 테스트 코드 포함
- 문서 업데이트 필요 시 포함
